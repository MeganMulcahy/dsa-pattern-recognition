# Pattern Drill

Learn DSA patterns and practice pattern recognition on live LeetCode problems.

- **Learn** — Cheat sheets and guides for 16 common patterns
- **Practice** — Random free LeetCode problems; pick the pattern, reveal the solution
- **Stats** — Session progress stored in your browser

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to GitHub, then import the repo on [Vercel](https://vercel.com/new). No environment variables required.
